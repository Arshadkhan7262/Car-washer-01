# Car Wash Pro - Admin Panel

Admin dashboard for managing the Car Wash Platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Update `.env` with your API base URL

4. Run development server:
```bash
npm run dev
```

The app will open at `http://localhost:3001`

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- React Router
- TanStack Query
- Tailwind CSS
- Shadcn/ui components
- Lucide React icons
